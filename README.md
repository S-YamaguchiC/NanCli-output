#### 1. vue-chartjsのインストール
VueでPieChartを使うためには、まず``vue-chartjs``を入れる必要があります  
そこで、まずは以下のコマンドでプロジェクト内にvue-chartjsをインスコします
```
$ cd {プロジェクトのパス}
$ npm install vue-chartjs chart.js --save
```
